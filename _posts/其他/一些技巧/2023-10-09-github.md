# 2023.10.09

## [Explorer](https://docs.github.com/en/graphql/overview/explorer)

### 查看仓库相关信息
#### repo-id
```GraphQL 
query {
  repository (name: "repo-name" , owner: "owner-name")  {
        id
  }
}
```
#### repo-database-id
```GraphQL 
query {
  repository (name: "repo-name" , owner: "owner-name")  {
        databaseId
  }
}
```
