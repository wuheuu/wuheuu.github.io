# 2023.10.16
## 进程迁移
为了会话的稳定性，将meterpreter shell迁移到其他不易被关闭进程上，此时不易查出木马文件，而且注入之后原来的木马程序如果被删除，会话权限也不会消失。
```bash
getpid    # 获取当前进程的pid
ps   # 查看当前活跃进程
migrate <pid值>    #将Meterpreter会话移植到指定pid值进程中，也就是进程注入
kill <pid值>   #杀死指定进程
```
